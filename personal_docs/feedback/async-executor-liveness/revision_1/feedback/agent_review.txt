SUMMARY
--------------------------------------------------------------------------------

This task requires debugging a multi-module Rust workload executor that
processes plan files describing work items with lanes, capacity limits,
parent-child hierarchies, and branch cancellation. The agent must fix bugs
across the codebase so that `run-matrix.sh` produces a correct, deterministic
audit JSON with ledger and journal side-files for six test cases. The test
suite contains 11 well-documented tests verifying trace ordering, branch
removal, CAP overflow scheduling, deduplication, and output consistency.
